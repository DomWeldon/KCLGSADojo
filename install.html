What is pygamezero?

pygamezero is a python package that allows us to create games and animation using just python.

Install on Mac or Windows (⬆️⬇️)

Instructions

If you are on the Geocomputation or GSA course, you likely have Anaconda installed...

How does it work? (⬆️⬇️)

Create a new folder called dojo (or something similar)

Save the contents below as intro.py in the new folder 


    WIDTH = 300  # adjust these to make the screen bigger
    HEIGHT = 300

    def draw():
        screen.fill((128, 0, 0))

Now run the command: (see next slide ⬇️ if doesn't work) 


  pgzrun intro.py

You'll need to be in the same directory as the intro.py file.

On windows it might look something like this:


  (base) C:\Users\YourUsername>cd Documents\Coding\dojo
  (base) C:\Users\YourUsername\Documents\dojo>pgzrun intro.py

Remember, this is in the terminal/console (e.g. Anaconda Prompt), not python.

### Use these commands to navigat to the folder * `cd` = change directory, so `cd dojo` changes into `dojo` folder * `cd ../` = go up one folder * `ls -lah` list all files and folders in current directory (Mac) * `dir` list all files and folders in current directory (Windows)

Now check out the cool stuff you can do ➡️

Cool stuff!

Check out the pgzero documentation for cool things to do with the library. For instance, you could:

  • Put in a background
  • Play sound
  • Make images move around the screen
  • Make an interactive element when a user clicks on something

Getting Started

Workflow

To edit your game, you'll need to:

  1. Make your changes to intro.py
  2. Save intro.py
  3. Close and re-open pgzrun (Ctrl + C, then re-run command)

To edit a intro.py, use a text editor like notepad or WordPad. Alternatively, you may wish to download a special code editor, like Atom, VS Code.
